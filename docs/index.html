Add old pending balances (receivables & payables) when migrating to this system

How to Use Opening Balance

  • Receivables: Money that customers owe you from old sales
  • Payables: Money that you owe to suppliers from old purchases
  • These entries will appear in ledger accounts as "Opening Balance"
  • Use the date when you're starting to use this system (or your financial year start date)
  • These entries won't affect inventory or sales/purchase statistics
